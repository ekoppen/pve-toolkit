# CLAUDE.md — pve-toolkit

## Project overview

Proxmox VM automation toolkit. Creates cloud-init based VMs from templates on Proxmox VE hosts using bash scripts.

- **Repo:** `ekoppen/pve-toolkit` (previously `proxmox-templates`)
- **Target platform:** Proxmox VE (Debian-based), runs as root
- **Language:** Bash (4+), uses associative arrays
- **i18n:** All user-facing strings use `MSG_*` variables from `lib/lang/{en,nl}.sh`

## Structure

```
├── setup.sh              # First-time config (SSH keys, template, storage)
├── install.sh            # Installs scripts/snippets on Proxmox host
├── lib/
│   ├── common.sh         # Colors, logging, whiptail helpers, validation
│   ├── config.sh         # Generated by setup.sh (LANG_CHOICE)
│   ├── defaults.sh       # VM type registry (register_type)
│   ├── secrets.sh        # Secret generation
│   └── lang/             # en.sh, nl.sh — all MSG_* variables
├── scripts/
│   ├── create-template.sh  # Debian cloud template (supports --version 12|13)
│   ├── create-vm.sh        # Create VM from template
│   ├── menu.sh             # Interactive whiptail menu (pve-menu)
│   └── ...                 # Other VM management scripts
└── snippets/             # Cloud-init YAML configs per VM type
```

## Key conventions

- **i18n:** Never hardcode user-facing strings. Add `MSG_*` variables to both `lib/lang/en.sh` and `lib/lang/nl.sh`.
- **Type registry:** New VM types go in `defaults.sh` via `register_type()` + a matching snippet YAML.
- **UI:** Interactive menus use whiptail via helpers in `common.sh` (`menu_select`, `radio_select`, `confirm`, `input_box`).
- **Scripts run on Proxmox as root.** Installed to `/root/scripts/`, libs to `/root/lib/`, snippets to `/var/lib/vz/snippets/`.
- **No external dependencies** beyond what Proxmox provides (qm, wget, whiptail, etc).

## Development notes

- Test bash syntax with `bash -n <file>` before committing.
- The `--auto` flag means non-interactive (no prompts), used by menu.sh.
- Cloud images come from `cloud.debian.org`. Currently supports Debian 12 (Bookworm) and 13 (Trixie).
